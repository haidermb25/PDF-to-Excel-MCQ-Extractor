PDF-to-Excel MCQ Extractor

A desktop application built with Python and Tkinter that extracts MCQs from PDF files and converts them into well-structured Excel spreadsheets.

Features
PDF MCQ extraction
Automatic question parsing
Option extraction
Answer detection
Explanation extraction
Radio/Checkbox identification
Image support
Excel file generation
Batch processing (100 MCQs per file)
User-friendly desktop interface
Technologies Used
Python
Tkinter
PDF Processing Libraries
OpenPyXL / Pandas
Regular Expressions (Regex)
Excel Output Structure
Column	Description
Question	MCQ statement
Option A	First option
Option B	Second option
Option C	Third option
Option D	Fourth option
Answer	Correct answer
Explanation	Answer explanation
Question Type	Radio or Checkbox
Image	Associated image path/reference
How It Works
Select a PDF file.
The application reads and parses MCQs.
Questions, options, answers, explanations, and images are extracted.
Data is organized into a structured format.
Excel files are generated automatically.
Each Excel file contains up to 100 MCQs.
